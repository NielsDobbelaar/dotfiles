## Getting Started

### Prerequisites

- Install brew and node/npm

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/NielsDobbelaar/dotfiles
   ```
2. run the installs script
   ```sh
   ./installs.sh
   ```
3. run the configs script
   ```sh
   ./configs.sh
   ```
4. reopen terminal for configs to take effect
